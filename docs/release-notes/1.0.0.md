# Kioku 1.0.0 Release Notes

**Release Date:** February 2026

Kioku is a local-first flashcard and quiz application designed for studying and testing your knowledge. All your data is stored securely on your device.

---

## Flashcard Decks

Create and manage flashcard decks for effective studying.

- **Card Management** - Add, edit, and delete cards with front and back content
- **Text & Code Support** - Cards support plain text or code with syntax highlighting
- **Syntax Highlighting** - Over 30 programming languages supported including JavaScript, TypeScript, Python, Rust, Go, Java, C++, SQL, and more
- **Tags** - Organize cards with custom tags for flexible filtering
- **Shuffle Mode** - Optionally randomize card order when studying
- **Notes** - Add additional notes or hints to any card

---

## Study Mode

An immersive study experience with intuitive controls.

- **Flip Cards** - Click, tap, or press Space/Enter to reveal answers
- **Keyboard Navigation** - Use arrow keys to move between cards
- **Swipe Gestures** - Swipe left/right on touch devices to navigate
- **Tag Filtering** - Study specific subsets of cards by tag
  - **Any Mode** - Show cards matching any selected tag
  - **All Mode** - Show cards matching all selected tags
- **Progress Tracking** - Visual progress bar and card counter
- **Session Statistics** - Track study time and cards reviewed

---

## Listen Mode

Audio-based learning with text-to-speech.

- **Auto-Read Cards** - Have card content read aloud automatically
- **Hands-Free Study** - Learn while multitasking or on the go
- **Configurable Playback** - Control reading speed and voice settings

---

## Quizzes

Create quizzes to test your knowledge with immediate feedback.

- **Multiple Choice** - Create questions with 2-6 answer options
- **Fill in the Blank** - Text input questions with exact match checking
- **Multiple Answers** - Support for "select all that apply" questions
- **Explanations** - Add explanations to help understand correct answers
- **Question Shuffling** - Randomize question order for each attempt
- **Tags** - Organize questions by topic with custom tags
- **Code Questions** - Include code snippets with syntax highlighting

---

## Quiz Results & Scoring

Detailed feedback after each quiz attempt.

- **Instant Scoring** - See your score immediately after completion
- **Answer Review** - Review all questions with correct/incorrect indicators
- **Score History** - Track your performance over multiple attempts
- **Color-Coded Results** - Green (80%+), Yellow (60-79%), Red (<60%)

---

## Import & Export

Easily share and backup your content.

- **JSON Import** - Import decks and quizzes from JSON files
- **Drag & Drop** - Drop files directly into the app to import
- **Single Export** - Export individual decks or quizzes as JSON
- **Bulk Export** - Export multiple items as a ZIP archive with organized folders
- **Schema Documentation** - Full JSON schema reference in the Help section

---

## Statistics Dashboard

Track your learning progress over time.

- **Study Sessions** - Number of study sessions per deck
- **Study Time** - Total time spent studying each deck
- **Cards Reviewed** - Count of cards studied
- **Quiz Attempts** - Number of times each quiz was taken
- **Best & Average Scores** - Track quiz performance
- **Last Activity** - See when you last studied each deck or took each quiz

---

## User Profiles

Support for multiple users on a single device.

- **Local Profiles** - Create multiple user profiles
- **Isolated Data** - Each user has their own decks, quizzes, and statistics
- **Easy Switching** - Quick profile switching from the user menu

---

## User Interface

A clean, focused interface designed for learning.

- **Dark Theme** - Easy on the eyes with a Monokai-inspired color palette
- **Responsive Design** - Works on various screen sizes
- **Keyboard Shortcuts** - Efficient navigation without leaving the keyboard
- **Toast Notifications** - Non-intrusive feedback for actions
- **Help Documentation** - Built-in help with usage guides and import schemas

---

## Technical Details

- **Local-First** - All data stored locally in SQLite database
- **Cross-Platform** - Built with Tauri for Windows, macOS, and Linux
- **Privacy Focused** - No cloud sync, no tracking, no account required
- **Fast & Lightweight** - Native performance with minimal resource usage

---

## Getting Started

1. Create a user profile when you first launch the app
2. Create your first deck or quiz from the navigation menu
3. Add cards or questions to your content
4. Start studying or take a quiz to test your knowledge
5. Track your progress in the Statistics section

For detailed usage instructions, visit the **Help** section within the app.
