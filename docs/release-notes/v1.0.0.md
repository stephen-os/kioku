# Kioku 1.0.0

**Release Date:** February 2026

Kioku is a flashcard and quiz application designed for easy content sharing and progress tracking.

---

## Features

### Flashcard Decks
- Create decks with text or code cards (30+ languages with syntax highlighting)
- Organize cards with custom tags
- Add notes and hints to cards
- Optional shuffle mode

### Study Mode
- Flip cards with click, Space, or Enter
- Navigate with arrow keys or swipe gestures
- Filter by tags (Any/All matching modes)
- Track study time and progress

### Listen Mode
- Text-to-speech for hands-free studying
- Configurable playback speed

### Quizzes
- Multiple choice and fill-in-the-blank questions
- Support for multiple correct answers
- Add explanations for answers
- Code snippets with syntax highlighting

### Quiz Results
- Instant scoring with answer review
- Score history across attempts
- Color-coded results (green/yellow/red)

### Import & Export
- JSON import/export for decks and quizzes
- Drag & drop file import
- Bulk export as ZIP archive

### Statistics
- Study sessions, time, and cards reviewed per deck
- Quiz attempts with best/average scores

### User Profiles
- Multiple local profiles with isolated data
